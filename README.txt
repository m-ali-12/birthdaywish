Birthday Sister Website - Vercel Ready

Files:
- index.html
- styles.css
- script.js
- vercel.json

How to deploy on Vercel:
1. Unzip this folder.
2. Upload the folder to a GitHub repository OR drag-and-drop it into Vercel.
3. Framework preset: Other / Static Site.
4. Deploy.

To customize:
- Open index.html
- Replace "Dear Sister" with your sister's name.
- Replace "Your Brother" with your name.
